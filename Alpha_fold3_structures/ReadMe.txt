This folder will contain the scripts for running simulations of wago-3 without RNA using the AlphaFold3 structures.
The file "wago3_AF3_simulation_setup.sh" contain the script to make the setup and perform minimization.
The file "wago3_AF3_simulation.sh" contains the script to perform equilibration and production.
The file "wago3_AF3_simulation_script.sh" contains the for loop to execute the "wago3_AF3_simulation.sh" inside each folder for different Wago3_AF3_(i).pdb structures.
